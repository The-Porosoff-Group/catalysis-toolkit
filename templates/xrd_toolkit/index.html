Phase-card guide: Size refines that phase's isotropic crystallite-size broadening. Mustrain refines that phase's microstrain broadening. PO controls March-Dollase preferred orientation: off disables it, fixed applies the entered axis/value without refining, and refined lets the value vary. axis is the preferred-orientation direction as h k l. value is the March-Dollase ratio, where 1.0 means no texture. Fixed values should come from a validated prior or repeated comparison workflow. Uniform cell, shown only for W2C-like phases, preserves starting axis ratios while keeping the refined volume. It is off in the WC/W2C preset because it changes the fit and is meant for explicit diagnostic comparisons.

Run Refinement

Presets
GSAS-II controls:
Control guide: Quick constrained fit refines only background, phase scales, sample position, and shared peak width. Refine cell too adds unit-cell refinement while keeping atoms/Uiso/size fixed. Phase isolation uses GSAS-II calculated phase curves for the plotted phase envelopes. Refine Zero swaps the sample-displacement correction for a zero-shift correction. Free X allows Lorentzian strain broadening. Fix Y holds the Lorentzian size-width parameter at the entered centidegree value. Y ≥ 0 prevents unphysical negative size broadening. Refine Uiso lets atomic displacement factors refine. Refine XYZ lets atom coordinates refine and should usually stay off unless the structure is trusted and the data justify it. WC/W2C Synergy-S production preset ticks Quick constrained fit, Refine cell too, Phase isolation, Refine Zero, Free X, and Y ≥ 0; it sets WC PO to fixed [001] at 0.905 and leaves W2C uniform-cell scaling off unless you tick it in the W2C card.
GSAS-II refinement fits the measured XRD pattern using the queued CIF structures, shared instrument/profile terms, and per-phase scale/cell/HAP settings. CIF tick marks and phase envelopes are display outputs from the same phase list used for refinement.
